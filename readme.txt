# Activity 4 - Student Teacher management sysytem

## Project Description
This project contains two Python scripts, `students.py` and `teachers.py`, which are part of a simple school management system. The `students.py` script handles operations related to student information, while the `teachers.py` script manages teacher data.

## Setup
To set up this project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/KawyaPerera/lab04.git
    cd lab04
    ```

2. Ensure you have Python installed on your system. 

3. Run the scripts:
    ```bash
    python students.py
    python teachers.py
    ```

## Usage
### students.py
This script allows you to manage student information, including adding, removing, and updating student details.

### teachers.py
This script provides functionalities to manage teacher data, such as adding, removing, and updating teacher information.

## Branches
- `main`: The main branch containing the initial setup and core functionalities.
- `feature/add-student-removal`: A branch where the functionality to remove students was added.
- `feature/add-teacher-removal`: A branch for adding the functionality to remove teachers.
- `feature/update-student-info`: A branch for updating student information.
